## Install

AWS CLI - Version 1

`pip install awscli`

AWS CLI - Version 2

https://docs.aws.amazon.com/cli/latest/userguide/install-cliv2-linux.html

```bash
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```
